# GTM30 Bootcamp

Personal learning log for **Project GTM30** — a 30-day hands-on program to build the instincts and systems-thinking of a GTM (Go-To-Market) Engineer.

## About this repo

This repo tracks my daily progress through the bootcamp: notes from mentor reviews, architecture diagrams for the workflows we design, and the automation exercises I complete each day.

## Background

Coming from a Customer Success & Implementation background, I'm using this bootcamp to move from thinking about individual GTM tools (CRM, enrichment, AI) toward designing full **revenue systems** — pipelines that automate the repetitive, data-driven work so humans can focus on relationship-building and strategic decisions.
